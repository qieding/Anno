## Setup

1. Download [latest Version](https://git.io/annomods)
2. Unzip it
3. Run **ModManager.exe**

## 'Windows protected your PC'

1. When you receive this warning - **don’t click** on the *Don’t run* option
2. Instead - click on the **More info** link - displayed beneath the message
3. A new prompt will be displayed - just click **Run anyway**

## Hints

* Read the Mod descriptions (**RTMD!**)
* Choose wisely - because some Mods exclude each other
* Don't use Mod Manager while Anno 1800 is running
* Switch to **Loader Disabled** before Anno gets patched
* **Start Anno 1800** can take a while - be patient

## Troubleshooting

* Download and install [Visual C++ Redistributable Packages](https://aka.ms/vs/16/release/VC_redist.x64.exe)
* Try **Run as administrator** (UAC)
